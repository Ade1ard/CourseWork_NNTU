# 🌌 Console Space Dodger

Динамичная и хардкорная аркадная игра на чистом C, работающая прямо в командной строке Windows. Управляйте космическим кораблем, уворачивайтесь от процедурно генерируемых астероидов, рискуйте ради комбо-очков и продержитесь как можно дольше!

Игра использует низкоуровневые возможности Win32 API для отрисовки графики без мерцания, обеспечивая стабильные 100 FPS в полноэкранном режиме.

---

## 🔥 Ключевые особенности

* Высокая производительность: Двойная буферизация экрана (RenderChanges) — обновляются только изменившиеся символы, что исключает раздражающее консольное мерцание.
* Система комбо за риск (Maneuver System): Чем ближе вы пролетаете мимо астероида (но не врезаетесь), тем выше поднимается ваш множитель комбо и быстрее растут очки.
* Динамическая сложность: Каждые 5 секунд игра увеличивает скорость падения камней, их максимальное количество на экране и частоту их появления. 
* Процедурные астероиды: Каждый камень имеет случайный размер, уникальную текстовую текстуру (форму), скорость по вертикали и может лететь по диагонали.
* True Fullscreen: При запуске игра автоматически переключает консоль в честный полноэкранный режим для максимального погружения.

---

## 🎮 Игровой процесс и управление

Ваша цель — выжить в пиксельно-текстовом космическом потоке. У вас есть 3 жизни, которые отображаются в виде сердечек.

* Движение (W, A, S, D / Стрелочки): Управление кораблем. По горизонтали корабль перемещается быстрее для эффективных маневров.
* Перезапуск (Пробел): Быстрый рестарт после экрана Game Over.
* Выход (ESC): Мгновенное закрытие игры.

## 🛠 Технологический стек

* Язык: C (C99 / C11)
* Платформа: Windows (используются библиотеки <windows.h>)
* Отрисовка: Символьная графика высокой плотности (разрешение консоли: 209 x 56 символов).
* Тайминги: Стабильные 100 FPS с привязкой к deltaTime (скорость игры не зависит от мощности процессора).

---

## ⚙️ Структура проекта

* main.c — основной игровой цикл, обработка ввода и дельта-тайма.
* Rocks.c / .h — логика поведения астероидов, динамический массив (RockList) и менеджер сложности.
* View.c / .h — кастомный движок отрисовки, работа с цветами консоли, вывод GUI (очки, комбо, жизни, таймер).
* Constants.h — конфигурация игры (FPS, размеры экрана, стартовые скорости).
